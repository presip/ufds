# UFDS — Universal Food Delivery Standard

OpenAPI 3.0 specification for integrating food delivery platforms
(Swiggy, Zomato, ONDC) with cloud kitchens and restaurant systems.

- `ufds_v1.0_openapi.yaml` — Machine-readable OpenAPI 3.0.3 spec
- `UFDS_v1.0_Specification.docx` — Human-readable specification document

## Quick Start
Import `ufds_v1.0_openapi.yaml` into [Swagger Editor](https://editor.swagger.io) to explore the API interactively.
